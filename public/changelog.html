Changelog

The latest updates, improvements, and fixes shipped to the Canopy platform.

May 14, 2026

Streaming Webhooks v2

Real-time event delivery now supports batched payloads with configurable flush intervals. Reduces overhead by up to 80% for high-volume integrations.

New API
POST /v2/webhooks/stream → 200 OK (12ms)

Enhanced Merchant Intelligence

Our merchant identification model now achieves 97.2% accuracy across 14 additional countries. Coverage expanded to include LATAM and Southeast Asian markets.

Improved Enrichment

Dashboard Analytics Overhaul

Completely redesigned analytics dashboard with custom date ranges, exportable reports, and team-level usage breakdowns. Now 3x faster to load.

Improved Dashboard

Category Mapping API

New endpoint for custom transaction category mappings. Define your own taxonomy and let our enrichment layer automatically classify against it.

New API

Latency Improvements

P95 enrichment latency reduced from 145ms to 62ms through optimized model inference and edge caching. Particularly impactful for real-time fraud detection flows.

Improved Performance

Bug Fixes & Stability
